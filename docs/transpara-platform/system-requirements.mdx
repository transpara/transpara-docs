- **Operating System**:
  - Ubuntu Server 20.04 LTS or 22.04 LTS
  - Red Hat Enterprise Linux (RHEL) 8 or 9
  - SUSE Linux Enterprise / openSUSE 15 SP6
  
- **Storage**:

- **Memory**:
  - Preferred: 64GB RAM
  - Minimum: 16GB RAM (scalable as needed)
  
- **CPU**:
  - Preferred: 16 CPUs
  - Minimum: 8 CPUs (scalable as needed)
