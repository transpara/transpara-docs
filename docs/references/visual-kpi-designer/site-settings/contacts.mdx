The site settings described on this page affect the entire Visual KPI site. The configured settings make new contacts available in your Visual KPI instance. Users can refer to these contacts when seeking help, or you can use them when creating new KPI alerts. You find the Alerts configurations under Site Settings (<PngFont name='options28' />) in the Document Actions panel.
